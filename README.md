## ğŸš€ Installation Guide

1.  **Download:** Grab the latest `SquireVR.apk` from the [Releases](https://github.com/kyy1-cyy/Squire-VR/releases/latest) page.
2.  **Sideload:** Use SideQuest or a similar tool to install it to your Quest **one time**.
3.  **Permissions:** Go to `Settings > Apps > Squire VR > Permissions` and enable **All Files Access**. 
    * *This is required to move game files to your OBB folder!*
4.  **Play:** Find Squire VR in your **Unknown Sources** and start browsing.
